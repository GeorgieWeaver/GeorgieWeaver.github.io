I have created an agent-based model as part of my Core Programming for GIS module.

An agent based model is an iterative model, that contains agents within an environnment. The model then runs by allowing the agents to do what has been set up in the code, over a certain number of iterations. This then allows patterns to be shown, and within the model the complex interactions of the agents can be seen.

There are a number of characteristics a model has:

  • Builds agents in a space
  • Reads in environmental data
  • Gets agents to interact with the environment
  • Randomises the order of agent actions
  • Displays the model as an animation
  • Is contained within a GUI
